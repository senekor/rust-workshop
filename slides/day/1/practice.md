# Choose your `$PATH`

should be handled by the install script

Make sure you you can call `cargo`.\
If you can't, try some of these:

```bash {lineNumbers: false}
# POSIX shell
source ~/.cargo/env
# or
export PATH="$HOME/.cargo/bin:$PATH"
```

```sh {lineNumbers: false}
# fish
fish_add_path ~/.cargo/bin
```
