# `for` Loops

more details on day 3

```rust {1,4,7|1,4-7|1,9,11|all}
let a = [10, 20, 30, 40, 50];

// `..` is the range operator
for i in 0..a.len() {
    let element = a[i];
    println!("the value is: {}", element);
}

for element in a {
    println!("the value is: {}", element);
}
```

# Off-Topic: Operators

some assignment variants exist ( `+=` )

| Comparison | `==` `!=` `<` `<=` `>` `>=`      |
| ---------: | :------------------------------- |
| Arithmetic | `+` `-` `*` `/` `%`              |
| Boolean    | `&&` `\|\|` `!`                  |
| Bitwise    | `&` `\|` `^` `!` (no tilde!)     |
| Range      | `..` `..=` (integers and `char`) |
